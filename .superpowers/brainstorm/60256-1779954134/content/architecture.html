Парсер (ночной запуск):
1. Playwright открывает house.kg + lalafo.kg
2. Собирает объявления по фильтру "Бишкек, земельные участки"
3. Для каждого объявления: ID, цена, площадь, район, адрес, дата
4. Сравнивает с предыдущими данными → фиксирует изменения цен
5. Вычисляет цену за сотку, среднюю по району
