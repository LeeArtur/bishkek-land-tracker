⭐ Логика рекомендаций

💰 Выгодная сделка
Объявление помечается как "выгодное" если:
price_per_sotka < district_avg × 0.85

т.е. цена за сотку на 15%+ ниже средней по району.

Порог 15% можно настраивать в конфиге.
📉 Тренд по районам
Каждую ночь после парсинга:
1. Берём все активные объявления по району
2. Считаем медианную цену/сотка за день
3. Записываем в price_history
4. API отдаёт историю за 30/90/365 дней

Медиана устойчивее среднего к выбросам.
🔔 Снижение цены
При каждом парсинге сравниваем
новую цену с предыдущей.
Если изменилась — пишем запись
в price_history с change_pct.

Дашборд показывает объявления
с изменением цены за сегодня.
